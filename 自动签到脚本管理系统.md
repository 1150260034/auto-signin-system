# 自动签到脚本管理系统

## Core Features

- 多账号Cookie管理

- 抓包教程指导

- 定时自动签到

- 签到状态监控

- 执行日志记录

## Design

现代简约的Material Design风格，以白色为主背景配合蓝色主色调，采用卡片式布局和清晰的信息层级设计，提供直观的账号管理、定时任务配置和日志查看界面
